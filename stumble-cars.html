Stumble Cars

Stumble Cars is a multiplayer racing game inspired by Stumble Guys, featuring chaotic obstacle course racing with up to 8 players simultaneously. Built with Unity and Photon networking for seamless multiplayer experiences. Players navigate through challenging obstacle courses while competing against each other in real-time, creating an exciting and unpredictable racing environment.

Project Type Multiplayer Racing Game Technologies Unity, C#, Photon PUN Platform Multi-platform

Game Features

Multiplayer Racing

Real-time multiplayer racing with up to 8 players, featuring synchronized gameplay, player interactions, and competitive leaderboards.

Obstacle Course Design

Challenging obstacle courses with dynamic elements, power-ups, and environmental hazards that create unpredictable and exciting gameplay.

Character Customization

Extensive character customization system with unlockable skins, accessories, and visual upgrades to personalize the racing experience.

Cross-Platform Support

Built for multiple platforms with optimized performance and consistent gameplay experience across different devices.

Technical Implementation

Networking Architecture

Implemented using Photon PUN for real-time multiplayer synchronization, ensuring smooth gameplay with minimal latency and robust connection handling.

Performance Optimization

Optimized rendering pipeline, efficient physics calculations, and streamlined networking code to maintain 60 FPS across all platforms.

Development Challenges & Solutions

Multiplayer Synchronization

Challenge: Maintaining consistent game state across multiple players with varying network conditions.
Solution: Implemented client-side prediction with server reconciliation, ensuring smooth gameplay even with network latency.

Physics Optimization

Challenge: Complex obstacle interactions causing performance issues with multiple players.
Solution: Developed custom physics layers and optimized collision detection systems for better performance.
